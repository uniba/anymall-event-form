Pattern 01
Classic 3-Column Grid
クリーンな3カラムグリッドで会場を横並び表示。レスポンシブで3→2→1列に変化。繊細なボーダーとシャドウのカードデザイン。
横並び ペールピンク カード型
02
Pattern 02
Magazine Editorial
ELLE風エディトリアルレイアウト。大きなヒーローエリア、2カラム構成、豊富な余白。ファッション誌のような洗練されたデザイン。
横並び ローズ エディトリアル
03
Pattern 03
Vertical Stack Cards
縦積みフルワイドカード。各会場がフル幅で表示され、件数が増えても崩れないレイアウト。ラベンダー基調の上品な配色。
縦積み ラベンダー フルワイド
04
Pattern 04
Alternating Zigzag
画像とテキストが左右交互に配置されるジグザグレイアウト。自然をイメージしたミント/セージの配色でペットテーマにマッチ。
縦積み ミント/セージ ジグザグ
05
Pattern 05
Minimal Card
超ミニマルデザイン。細い1pxボーダー、シャドウなし、豊富な余白。ホバーで詳細表示。ハイファッション的な美学。
横並び ホワイト ミニマル
06
Pattern 06
Tab by Area
エリア別タブ切替（すべて/東京/大阪）で会場をフィルタリング。直感的なUI、スムーズな切り替えアニメーション。
横並び ピーチ タブ切替
07
Pattern 07
Timeline Layout
縦タイムラインで会場を時系列表示。中央線の左右に交互配置。イベントの流れが視覚的に伝わるエレガントな構成。
縦積み アイボリー タイムライン
08
Pattern 08
Masonry Grid
Pinterest風の段違いグリッド。カードの高さが異なり視覚的に変化を与える。CSSカラムによるメイソンリーレイアウト。
段違い ブルーグレー メイソンリー
09
Pattern 09
Full-Width Band
各会場をフルワイド横帯で表示。交互に背景色を変え、画像とテキストを横並びに配置。縦スクロールに最適。
縦積み アプリコット フルワイド帯

Pattern 10
Asymmetric Grid
非対称グリッドで視覚的な動きを表現。CSS Gridのテンプレートエリアを活用。大小混在のカードレイアウト。
非対称 プラム CSS Grid
